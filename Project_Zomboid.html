Project Zomboid

Project Zomboid — симулятор выживания с элементами RPG, в котором игрок пытается пережить конец света. Игра разработана студией The Indie Stone. В обучении тебя жестко разводит енот-макскот данной игры, то есть уже с обучения тебя учат умирать.
